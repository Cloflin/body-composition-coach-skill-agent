# Verified authority sources & sourcing protocol (Tier 1 / Tier 2)

This file holds the **verified base** of real sources for the skill, split into two
tiers, plus the protocol for keeping them current via web search. The whole point
of the skill is to never invent facts or links, and to never pass non-government
science off as a WHO/ministry standard — so sources are real and tier is explicit.

## The no-fabrication rule (restate before every plan)

- Cite only real pages. Never invent a URL, DOI, study, statistic, or quote.
- If web search is available, confirm the page is live and current before citing.
- If you cannot ground a specific number or claim, give the general principle
  instead, or omit it. Do not fill the gap with a plausible-sounding invention.
- **Always tag the tier.** Tier 2 claims must be marked "not a WHO/ministry source".

## TIER 1 — WHO and national health ministries/agencies (primary authority)

### International — World Health Organization (WHO)
- Physical activity (fact sheet) — recommended activity levels, benefits.
  https://www.who.int/news-room/fact-sheets/detail/physical-activity
- WHO guidelines on physical activity and sedentary behaviour (2020).
  https://www.who.int/publications/i/item/9789240015128
- Healthy diet (fact sheet) — principles, limits on free sugars/fat/salt.
  https://www.who.int/news-room/fact-sheets/detail/healthy-diet
- Obesity and overweight (fact sheet) — definitions, BMI, drivers, risks.
  https://www.who.int/news-room/fact-sheets/detail/obesity-and-overweight

### United Kingdom — NHS / NICE
- NHS weight loss (incl. the safety stance: not for children/pregnant; consult GP
  if you have a condition). https://www.nhs.uk/better-health/lose-weight/
- NHS Eatwell Guide — balanced-diet food-group proportions.
  https://www.nhs.uk/live-well/eat-well/food-guidelines-and-food-labels/the-eatwell-guide/
- NHS Obesity — states BMI's limitation (muscular people can have high BMI without
  excess fat) and recommends waist-to-height ratio as an additional measure.
  https://www.nhs.uk/conditions/obesity/
- NHS waist-to-height ratio (keep waist < half your height; WHtR < 0.5). Robust to
  high muscle mass per NICE.
  https://www.nhs.uk/health-assessment-tools/calculate-your-waist-to-height-ratio

### United States — USDA / HHS
- Dietary Guidelines for Americans — landing page always points to the current
  edition (now 2025-2030). https://www.dietaryguidelines.gov/

### Additional Tier-1 authorities — find the exact current page via search
- CDC (US) — Healthy Weight, physical-activity basics. Domain: cdc.gov
- HHS (US) — Physical Activity Guidelines for Americans. Domain: health.gov
- The **user's own national health ministry/agency** — find via search (protocol).

## TIER 2 — peer-reviewed scientific consensus (NOT WHO/ministry; label as such)

Use only where Tier 1 is silent (muscle-building protein, hypertrophy programming,
body-fat-% ranges). Reputable and citable, but not government/WHO — always say so.

- **Protein for muscle:** International Society of Sports Nutrition (ISSN) Position
  Stand: Protein and Exercise (Jäger et al., 2017, J Int Soc Sports Nutr 14:20).
  Building/maintaining muscle: ~1.4-2.0 g/kg/day for most exercising individuals;
  ~2.3-3.1 g/kg may help preserve lean mass during a calorie deficit; per-meal dose
  ~0.25 g/kg or ~20-40 g. https://pmc.ncbi.nlm.nih.gov/articles/PMC5477153/
- **Body-fat-% ranges:** no crisp Tier-1 cut-offs exist. If the user wants a target,
  attribute any range to the body that publishes it (e.g. ACE/ACSM), confirm via
  search, and note it is not a WHO/ministry standard. Prefer tracking the trend +
  WHtR over asserting a fixed "healthy %".

## Key figures (confirm against the live source before stating)

- Healthy BMI range ~18.5-24.9; waist-to-height ratio target < 0.5 (Tier 1).
- Adults: ~150-300 min/week moderate aerobic (or 75-150 vigorous) + muscle-
  strengthening >=2 days/week (Tier 1, WHO).
- Free sugars <10% energy (ideally <5%); total fat <30%; saturated fat <10%; salt
  <5 g/day (Tier 1, WHO).
- Safe fat-loss rate ~0.25-1 kg/week; 5-10% body-weight loss gives meaningful
  benefit (Tier 1, NHS/CDC).
- Muscle-building protein ~1.4-2.0 g/kg/day (Tier 2, ISSN — not WHO/ministry).

These were current at last update. Editions change — prefer confirming live.

## Web-search actualization protocol (when search is available)

1. **Confirm currency.** Search the authority's site for the current edition/fact
   sheet and use the live page and its current numbers.
2. **Find the user's national source.** If the country is known, search its official
   ministry/agency guidance and prefer it alongside WHO. Don't guess the URL.
3. **Prefer authoritative domains.** International: who.int. Government/agency:
   *.gov, *.gov.uk, nhs.uk, nice.org.uk, and the user's national ministry domain.
   For Tier 2, prefer the original position stand / peer-reviewed source. Avoid
   blogs, content farms, and commercial diet sites.
4. **Cite what you used, with tier tags.** Real, live links only.
5. **No-search fallback.** Rely only on this base, use only the figures above, and
   tell the user you're limited to the bundled sources rather than inventing.
